## Installation

```bash
$ git clone git@gitlab.com:dimasjt/sinatra-angularjs.git

$ cd sinatra-angularjs

$ bundle install

$ rackup

```

## Technologies

- Sinatra
- AngularJS
- Restangular
- Bootstrap
- Coffeescript

- Lodash
